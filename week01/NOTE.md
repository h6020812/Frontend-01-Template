# 重學｜工程體系

關於工程面的問題，不單單只有技術層面，而是在於多方領域一起解決問題。這樣在成就感來說，就不會只有技術成就。

===數據驅動思考方式===

目標 => 現狀 => 方案 => 實施 => 結果

1. 分析業務目標，訂下數據指標
2. 採集數據，建立數據展示系統
3. 設計技術方案，預估數據
4. 小規模實驗，推廣全公司落地，形成制度
5. 統計最終效果，跟老闆匯報
